These are the source files for https://chafc-fcig.github.io/

If you are a member of FCIG you can edit these files and the changes will be
reflected on the webpage. If you are a member of FCIG and are not able to edit
any file you want to, make sure you have a github account and we can get the
permissions correct so that you can edit.

The majority of the files and folders you can see above are related to the
building of the site. The file you might want to edit are:

- The
[index.md](https://github.com/chafc-fcig/chafc-fcig.github.io/blob/master/index.md)
file is the first page you see when you go to the website

- The
[README.md](https://github.com/chafc-fcig/chafc-fcig.github.io/blob/master/README.md)
file is this text you are reading now.

- The
[guides](https://github.com/chafc-fcig/chafc-fcig.github.io/tree/master/guides)
and
[discussions](https://github.com/chafc-fcig/chafc-fcig.github.io/tree/master/discussions)
folders contain files with the guides and discussions.

This site is built using [Jekyll](https://jekyllrb.com/). The documents are
written using [Github
Markdown](https://guides.github.com/features/mastering-markdown/)

There are instructions for how to edit files at
https://chafc-fcig.github.io/editing_this_site.html
